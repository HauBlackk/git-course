## Tổng hợp kiến thức 
1. Git
> Tạo repositories trên github <br>
Các câu lệnh để up code lên github <br>
**git inti** khởi tạo <br>
**git remote add origin link shh** liên kết với github <br>
**git add .** thêm chuẩn bị file dấu chấm đại diện cho tất cả <br>
**git commit -m"init project"** lưu lịch sử bản ghi <br>
**git push origin main** đẩy code lên github
